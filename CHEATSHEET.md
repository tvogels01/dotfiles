# Cheatsheet

## `conda`

```shell
conda create --name code 
conda info --envs

conda search
conda list
```

More at [cheatsheet.html](https://conda.io/projects/conda/en/latest/user-guide/cheatsheet.html)

## `npm` / TypeScript

```shell
npm install --global typescript
```

## `nvm`

```shell
nvm install --lts
nvm use --lts
```
