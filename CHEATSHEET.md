# Cheatsheet

## `conda`

```shell
conda create --name code 
conda info --envs

conda search
conda list
```

More at [cheatsheet.html](https://conda.io/projects/conda/en/latest/user-guide/cheatsheet.html)

## `nvm`

```shell
nvm install --lts
nvm use --lts --save
```

Then leverate the new `.nvmrc` file:
```shell
nvm use
```

## `npm` and TypeScript

Update `npm`:
```shell
npm install --global npm
```

```shell
npm install --global typescript
```
